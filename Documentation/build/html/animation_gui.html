animation_gui module¶

class animation_gui.Animation(sol, list_of_object_lists, ax)[source]¶

Bases: object

Class for creating an animation of a system of springs and masses.

Parameters:

  • sol (scipy.integrate.OdeSolution) – Solution object containing the time, position and velocity data

  • list_of_object_lists (list of lists of objects) – List containing lists of Spring and Mass objects

  • ax (matplotlib.axes.Axes) – Matplotlib axis object to plot the animation

Variables:

  • sol (scipy.integrate.OdeSolution) – Solution object containing the time, position and velocity data

  • list_of_object_lists (list of lists of objects) – List containing lists of Spring and Mass objects

  • ax (matplotlib.axes.Axes) – Matplotlib axis object to plot the animation

  • spring_lines (list of matplotlib.lines.Line2D) – List to store the plotted spring lines in the animation

  • mass_circles (list of matplotlib.patches.Circle) – List to store the plotted mass circles in the animation

  • sb_rectangles (list of matplotlib.patches.Rectangle) – List to store the plotted steady body rectangles in the animation

init_animation()[source]¶

Initialize the animation by setting up the plot with springs and masses in the starting configuration, which refers to the initial conditions of the Spring and Mass objects.

update_frame(num)[source]¶

Update the animation frame based on the current time step.

Parameters:

num (int) – Current time step index
