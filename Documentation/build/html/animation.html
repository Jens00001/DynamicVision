animation module¶

animation.animation(sol, list_of_object_lists, skip_sim_steps=150)[source]¶
Parameters:

  • sol – solution of the simulation of the system (See documentation of solve_ivp())

  • list_of_object_lists (list) – a list which contains lists of objects created in the system

  • skip_sim_steps (int) – value of how many simulated steps are skiped before the next update of the animation, in case the animation is to slow because of too many frames
